# Crovi: Gestion-de-tiempo
Crovi busca disminuír la dificultad para gestionar el tiempo de forma efectiva que poseen muchos estudiantes y trabajadores. Los usuarios podrán planificar sus tareas diarias con tan solo ingresar sus tareas en la bandeja de texto que se encuentra al iniciar la aplicación. A partir de ahí, se crea un espacio con dichas tareas, lo único que debe hacer el usuario es ingresar la fecha límite para completar sus tareas y como clasificaría la tarea usando un sistema de etiquetas de colores. <br/>
Crovi va dirigido a estudiantes y trabajadores con un horario exigente o bien que simplemente les interesa desarrollar un habito de planificación de tareas. Esto debido a que puede llegar a ser bastante útil para librar al usuario de esta fuente de estrés y así contrarrestar las altas exigencias y vivir con más tranquilidad. 

### Servicios:
* Firebase Authentication: Registro e inicio de sesión con email, Google y Facebook.
* Firebase Cloud Messaging (FCM): para notificaciones push en Android.
* Google Calendar API: Conexion con google calendar. <br/>
  
* Adicionalmente se trabajará con una base de datos local con tareas pre-creadas.
